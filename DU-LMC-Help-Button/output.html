Install And Setup

DO NOT ACTIVATE CENTER BOARD UNTIL YOU READ!
1: Place mining units by using the center of the red square.
2: Remove red squares.
3: Link mining units to storage hub.
3: Link mining units to "LEFT" programming board.
4: Turn all mining units on.
5: Check Options on storage board (right board) to match your stats for containers.

Script Error

If you see a script error on the calibration screen please be sure to have atleast one mining unit linked to it. All mining units that are hooked to it must be on.
